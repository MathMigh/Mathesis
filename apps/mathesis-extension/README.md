# Extensao Mathesis

1. Abra `chrome://extensions` ou `edge://extensions`.
2. Ative o modo de programador.
3. Clique em **Carregar sem compactacao** e selecione esta pasta.
4. Depois de editar a extensao, clique em **Atualizar** na pagina de extensoes.

Ao selecionar uma unica palavra em uma pagina comum, a extensao abre o Mathesis automaticamente.

Paginas internas do navegador, como `chrome://extensions`, e o leitor PDF nativo do Chrome/Edge nao permitem injecao de extensoes. Para PDFs, use o leitor do proprio Mathesis.

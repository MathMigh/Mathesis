# Extensao Mathesis

Esta extensao abre o painel do Mathesis a partir de uma selecao numa pagina.

1. Abra `chrome://extensions` ou `edge://extensions`.
2. Ative o modo de programador.
3. Escolha **Carregar sem compactacao** e selecione esta pasta.
4. Em arquivos locais, ative tambem **Permitir acesso a URLs de arquivo** nas opcoes da extensao.

A extensao le apenas a selecao no instante em que o botao Mathesis e acionado. Ela nao contem chaves de API: as consultas seguem para o servidor do Mathesis.

Paginas internas do navegador e o leitor PDF nativo do Chrome/Edge nao permitem injecao de extensoes. Para PDFs, use o leitor do proprio Mathesis.
